# 커뮤니티 게시판 재오픈 운영 안내

자유게시판, 장비 사용기, 중고거래 게시판은 `NEXT_PUBLIC_COMMUNITY_BOARDS_ENABLED` feature flag로 닫기/열기 상태를 제어합니다.

## 열기/닫기 방법

- 닫기: `NEXT_PUBLIC_COMMUNITY_BOARDS_ENABLED=false` 또는 환경변수 미설정
- 열기: `NEXT_PUBLIC_COMMUNITY_BOARDS_ENABLED=true`
- Vercel 환경변수를 변경한 뒤에는 사이트를 재배포해야 public UI와 API 라우트에 반영됩니다.

## 적용 범위

- flag가 `true`일 때만 자유게시판(`/board/free`), 중고거래(`/board/market`), 장비 사용기(`/board/gear`) public UI/API 접근을 허용합니다.
- flag가 `false` 또는 미설정이면 기존처럼 커뮤니티 게시판 접근은 중단 안내/차단 상태를 유지합니다.
- 공지사항, QnA, 리뷰는 고객센터/리뷰 성격으로 기존 흐름을 유지합니다.
- 홈 화면의 중고거래 최신글 프리뷰는 이 flag 변경만으로 자동 재도입하지 않습니다.

## 중고거래 운영 체크리스트

중고거래 게시판을 운영할 때는 아래 정책을 함께 확인해 주세요.

- 금지 품목: 도난품, 위조품/가품, 불법 복제품, 개인정보가 포함된 물품, 법령상 거래가 제한된 물품, 위험물·무기류·의약품·주류·담배 등 제한 품목
- 사기, 허위매물, 도난품, 위조품, 불법·위해 상품 등록 금지
- 거래 전 상품 상태, 가격, 배송 방식, 환불 가능 여부 확인 안내
- 문제가 있는 게시글은 신고 또는 관리자 판단에 따라 숨김/삭제될 수 있음을 안내
- 신고 접수, 관리자 숨김/삭제, 목록/인기글 노출 제외 흐름이 정상 동작하는지 운영 전 확인
